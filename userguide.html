Tool Tagging

Application User Guide

Application Overview

The Tool Tagging application will facilitate the tracking and maintenance of tool information on real time basis to imply Ford ownership and tag tools with Ford-issued tool tags. As a part of Ford Global Terms and Conditions and the Tooling Purchase and Audit guidelines, production tooling owned by Ford Motor Company and located on the premises of a supplier must be and remain, permanently tagged to reflect Ford's ownership.

Prerequisites & Assumptions

  • A mobile device with an active mobile data connection to access the Tool Tagging application and use its features.
  • The user should have a valid Covisint ID and access to the Ford Supplier Portal in order to access the Tool Tagging application.
  • The access device should have a working GPS and Camera hardware in order to support the scanning and upload features.
  • The user should be familiar with the camera interface of his/her Android/iOS device.
  • The user should be familiar with modern Android (Chrome) & iOS (Safari) web browsers.

Intended Users

  • Ford Suppliers with access to WebQuote.
  • Ford Suppliers identified by Ford Motor Company for using the Tool Tagging application.
  • Ford Suppliers with tool orders raised from WIPS.
  • Ford Suppliers aware of the tool tagging business processes within Ford Motor Company.

Application Workflows


Covisint Login / Logout

Title: Covisint Login page

Actions: Login button to authenticate login credentials and enter Tool Tagging application.

Possible Errors:

  • You have entered an invalid password and/or User ID, please try again.
  • Cause: Invalid user ID or password has been entered
  • Solution: Check and re-enter your User ID/password. **They are both case sensitive fields.

Application Home Page

Purpose

Provides access to the Tool Tagging web application after authentication via Covisint. Provides buttons to navigate through features and log out of the application.

Screen Workflow

Title: App Home page

Actions:

  • Order Tool Tag button to search for tools and order tags.
  • Upload Tool Photo button to take tool photos and upload them to WebQuote.
  • Scan Tool Tag button to scan tool tags and get additional tool details.

Possible Errors:

  • Page not responding
    • Cause: Page idle for long or slow internet connection.
    • Solution: Refresh page or re-login through Covisint.
  • Refer to general application errors (Appendix A) for other possible errors on this screen.

Order Tool Tag

Purpose

Provides features to:

  • Search for tools using Tool Order numberor Ford Asset Number.
  • Check tool description for the searched tool order number or Ford asset number.
  • Select individual/multiple tools for generating tool tags.
  • Enter and edit tag delivery addresses for one/multiple/all tools.
  • View history of address changes madeduring an activeuser session.
  • View component part numbers of assets while editing Tag Delivery Address.
  • Confirm order for tool tags to Ford Motor Company.

Title: Tool Order Search

Actions:

  • Search button to initiate search for the entered keyword (Tool Order number or Ford Asset Number)

Possible Errors:

  • Please enter Tool Order Number / Please enter Ford Asset Number.
    • Cause: Empty search field.
    • Solution: Enter a valid tool order number/ford asset number to search.
  • Refer to general application errors (Appendix A) for other possible errors on this screen.

Title: Tool Order Search Results

Actions:

  • Search button to initiate search for the entered keyword (Tool Order number or Ford Asset Number)
  • Checkboxesfor selecting assets
  • Select All/Select None button to select assets for which tags are needed
  • Show more/Show less for expanding/collapsing the asset description
  • Order Tag button for proceeding with ordering tags for the selected assets.
  • Tool Tag order/reorder indicators ! to indicate the number of tool tag orders for every asset.
    • Tool Tag for this asset has been ordered.
    • ! High volume of tag orders for this asset.
    • Tool tag orders disabled for this asset due to excessive tag orders.

Possible Errors:

  • Make a Selection
    • Cause: No asset selected for ordering tag
    • Solution: Select at least one asset for ordering tag
  • Refer to general application errors (Appendix A) for other possible errors on this screen.

Title: Tag Delivery Address

Actions:

  • Checkboxesfor selecting one or multiple assets to edit current tag delivery address.
  • Select All/Select None button to select/deselect all assets at once for edit.
  • View History button to display history (for that session) of edited addresses. Blank when no edits made.
  • Edit selected button to edit the delivery of selected (checked) assets.
  • Confirm button to submit request for Tool Tags to Ford Motor Company.
  • View History to view the last address for each asset prior to the most recent change in address.
  • Tag Size to choose a preferred tag size and material for each asset in the order.

Possible Errors:

  • Please complete all mandatory fields.
    • Cause: One or more mandatory address fields are missing.
    • Solution: Check each of the highlighted boxes and ensure that mandatory fields are complete.
  • Refer to general application errors (Appendix A) for other possible errors on this screen.

Title: Edit Delivery Address - for individual asset address change

Actions:

  • Mandatory Contact Name field to enter the name of the contact person authorized to receive tool tags from Ford.
  • Mandatory Street, Country, City & State fields for accurate delivery of Tool Tags shipment.
  • Save button to save the address change
  • Cancel button to discard all changes

Possible Errors:

  • Contact Name is required | Street is required | Country is required | City is required | State is required
    • Cause: One of the mandatory address fields is missing. All address fields are mandatory.
    • Solution: Check each address field and ensure that mandatory fields are complete.
  • Refer to general application errors (Appendix A) for other possible errors on this screen.

Title: Edit Delivery Address – for multiple assets address change

Actions:

  • Contact Name field to enter the name of the contact person authorized to receive tool tags from Ford.
  • Mandatory Street, Country, City, State & Postal Code fields for accurate delivery of Tool Tags shipment.
  • Save button to save the address change for all selected assets.
  • Cancel button to discard all changes.

Possible Errors:

  • Contact Name is required | Street is required | Country is required | City is required | State is required
    • Cause: One or more mandatory address fields are missing.
    • Solution: Check each address field and ensure that mandatory fields are complete
  • Refer to general application errors (Appendix A) for other possible errors on this screen.

Scan Tool Tag

Purpose

Provides features to:

  • Scan Ford-provided tool tags generated through the Tool Tagging application or WebQuote.
  • Validate the tool tag present on the tool to know the status of the tool and if any actions are required.
  • View additional details about the tool after successful validation of the tag.
  • Rescan a tag.

Title: Scan screen for Tool Tag QR code

Actions:

  • Scan Ford-provided tool tag QR code. Place the QR code within the highlighted region on the camera screen to scan.

Possible Errors:

  • Scan timed out
    • Cause: Scan screen idle for longer than 15 seconds
    • Solution: Try rescanning with tool tag QR code in the highlighted box on the camera screen
  • Refer to general application errors (Appendix A) for other possible errors on this screen.

Title: Success - Tool Tag Validated Successfully

Actions:

  • 1. Rescan button to navigate back to the QR code scanner.
  • 2. Details button to view additional information of the tool. (Refer screen 3 in Scan Tool Tag).

Possible Errors:

  • Refer to general application errors (Appendix A) for possible errors on this screen.

Title: Error! - Asset Number not found

Actions:

  • Rescan button to navigate back to the QR code scanner.

Possible Errors:

  • Asset Number not found. Please reference the Tool Tagging application user guide.
    • Cause: Scanned Ford Asset Number not found in WebQuote database. The tag might be damaged or QR code on the tag is no longer legible for scanning.
    • Solution: Try rescanning with tool tag QR code in the highlighted box on the camera screen.
  • Refer to general application errors (Appendix A) for other possible errors on this screen.

Title: Error! - Asset assigned to different supplier

Actions:

  • Rescan button to navigate back to the QR code scanner.

Possible Errors:

  • Asset Number assigned to different supplier.
    • Cause: Scanned Ford Asset Number has been assigned to a different supplier.
    • Solution: The tool tag needs to be reordered.
  • Refer to general application errors for other possible errors on this screen.

Title: Asset Moved!

Actions:

  • Rescan button to navigate back to the QR code scanner.
  • Details button to view additional information of the tool. (Refer screen 3 in Scan Tool Tag)
  • Location for this asset does not match Ford records. Please notify the Ford Buyer. - The asset has moved from the location where the tool tag was originally ordered from. Notify the Ford Buyer of this change, if not done already.

Possible Errors:

  • Refer to general application errors (Appendix A) forpossible errors on this screen.

Title: Action Required!

Actions:

  • 1. Rescan button to navigate back to the QR code scanner.
  • 2. Details button to view additional information of the tool. (Refer screen 3 in Scan Tool Tag)
  • 3. Asset location not in WebQuote, please provide the address in WebQuote. – The current tool location for the tool is not complete or absent in WebQuote. Please log in to WebQuote from a PC and update the current location for this asset.
  • 4. WebQuote URL:www.webquote.ford.com

Possible Errors:

  • Refer to general application errors (Appendix A) forpossible errors on this screen.

Title: Error! – Tool Tag Invalid

Possible Errors:

  • Tool Tag is invalid. Please reorder the tool tag.
    • Cause: Scanned tool tag is invalid. It might be damaged or no longer legible for scanning.
    • Solution: The tool tag needs to be reordered.

Appendix A


General Application Errors

Tool Tagging application is temporarily unavailable

  • Cause: The application is facing issues in interacting with the Ford network.
  • Solution:
    • Try refreshing the page
    • Close the current session and reopen the application in a new session in your browser. Wait for 5-10 minutes before accessing the application again.

Session Expired! Please login again to continue.

  • Cause: Application screen idle for too long.
  • Solution: Login again from the Covisint portal
